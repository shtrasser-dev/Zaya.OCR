Namespace Zaya.OCR.Impl.OneOcr.Services

Classes

OneOcrService

OneOCR engine implementation of IOCRService using P/Invoke into the native oneocr.dll (Windows 11 SnippingTool OCR engine). No WinRT, no Windows App SDK, no package identity required. Call InitializeAsync(IReadOnlyDictionary<string, object?>?, CancellationToken) before creating sessions.

OneOcrSession

OneOCR session that performs OCR using the native oneocr.dll via P/Invoke.
