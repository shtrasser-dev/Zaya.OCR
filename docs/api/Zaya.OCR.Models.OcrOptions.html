Class OcrOptions

Configuration options for OCR recognition operations. Options that are not supported by a particular engine implementation are silently ignored. When Width, Height, Stride, and PixelFormat are set, the byte[] passed to RecognizeAsync(byte[], CancellationToken) is treated as raw pixel data rather than an encoded image (PNG, JPEG, BMP).
